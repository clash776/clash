この章ではシリアルモニタの使い方について学びます。

シリアルモニタとは、マイコンとパソコン(マイコン同士も可能)との通信を利用したデータの送受信や
デバッグ(プログラムの不具合を修正する作業)などに利用されるものです。

  1. シリアル通信の準備

    2. 
            
  2. プログラムで通信開始

    3. 
            
  3. シリアルモニタに表示する

    4. 
          

        

        

          

            
Serial.begin関数

            シリアル通信を使うためには、Serial.begin関数で通信速度(ボーレート)を指定して、通信を開始する必要があります。

            
Serial.begin関数は以下のように記述します。

            
Serial.begin(通信速度);

            
例えば、通信速度を9600bpsに設定する場合、setup関数の中に以下のように記述します。

            
void setup() { Serial.begin(9600); // 通信速度を9600bpsに設定 }

          

          
注意：実際のエディタ(Arduino IDEなど)でシリアルモニタを使う場合、

            シリアルモニタ側の通信速度設定を、プログラムの値（例：9600）に合わせる必要があります。

        

        

          

            
Serial.print関数 / Serial.println関数

            シリアルモニタにデータを表示するには、Serial.print関数またはSerial.println関数を使います。

            
これらの関数は以下のように記述します。

            
Serial.print(表示したいデータ);
または
Serial.println(表示したいデータ);

            
例えば、シリアルモニタに「Hello」と表示する場合、loop関数の中に以下のように記述します。

            
void loop() { Serial.print("Hello");}
または
void loop() { Serial.printIn("Hello");}

            
Serial.print関数は改行せずにデータを表示し、Serial.println関数は改行してデータを表示します。

6章おまけ：学んだ機器が使われている場所

        
6章のおまけとして、これまで学んできた機器がどんなところで使われているかを紹介します。

        

          
マイコン

          

            
          

          
本教材ではArduinoを使ってマイコンの学習を行ってきましたが、

            実際の製品に組み込まれているマイコンは、Arduino本体のような形ではなく、

            もっと小さな部品として基板の上に直接取り付けられています。
          

          
小型のマイコンは、スマートフォン、家電製品、自動車、産業機器など、

            さまざまな製品に組み込まれており、製品の制御、データ処理、通信などを担当しています。

          

        

        

          
LED

          

            
          

          

            本教材ではLEDを使って、マイコンの出力制御を学習してきましたが、

            実際の製品に使われているLEDは、実験用の部品としてではなく、

            製品の一部として内部に組み込まれています。
          

          

            LEDは電気を流すことで光る小さな部品で、

            動作状態の表示、情報の通知、照明など、

            さまざまな目的で利用されています。
          

          

            スマートフォンや家電製品、自動車、機械設備などでは、

            LEDがマイコンによって制御され、点灯・消灯や点滅を行うことで、

            ユーザーに状態を分かりやすく伝えています。
          

        

        

          
7セグメントLED

          

            
          

          

            本教材では7セグメントLEDを使って、複雑なLED機器の制御方法を学習してきましたが、

            実際の製品では、7セグメントLEDは単体の実験用部品ではなく、

            表示装置の一部として機器の内部に組み込まれています。
          

          

            7セグメントLEDは、複数のLEDを組み合わせることで、

            数字や簡単な情報を分かりやすく表示できる表示部品です。
          

          

            電子レンジやデジタル時計、電卓、測定器などの機器では、

            マイコンがどのセグメントを点灯させるかを制御することで、

            時間や数値、動作状態などを表示しています。
